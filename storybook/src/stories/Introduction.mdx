# Althus Design System

Bem-vindo ao **Althus Design System** — a biblioteca de componentes e tokens de design que padroniza a experiência visual dos produtos Althus.

## O que você encontra aqui

- **Foundations** — Paleta de cores, tipografia e espaçamentos
- **Components** — Componentes React prontos para uso, com suporte a dark mode e acessibilidade

## Design Tokens

Todos os componentes utilizam [CSS custom properties](https://developer.mozilla.org/en-US/docs/Web/CSS/--*) definidas em `src/tokens/tokens.css`.
Importe o arquivo de tokens antes de usar qualquer componente:

```tsx
import '@althus/design-system/tokens.css';
```
